# Knowledge Vault

The agent's media-buying brain. The Performance Agent reads these on every analysis so its calls and teaching reflect current best practice, not stale habits. Curated + version-controlled; refresh the benchmarks/platform files quarterly.

- **media_buyer_playbook.md** — the senior-buyer operating philosophy, account structure, creative rules, and the symptom→cause→fix diagnostic tree.
- **metric_benchmarks.md** — what "good" looks like (CTR/CPM/CVR/CPA/ROAS), 2025–26, sourced.
- **metric_glossary.md** — plain-language definitions for every metric (one line each).
- **meta_platform_2026.md** — current platform mechanics (targeting deprecation, Advantage+, CBO, learning phase).

**Refresh cadence:** benchmarks + platform files quarterly (or when Meta ships a major change). Update the "Last refreshed" date and the change log in ARCHITECTURE.md.

This vault is **curated knowledge (L1)**. It pairs with the **Recursive Learning vault (`../learning/`, L4)** — what actually worked for *our* brands — which overrides these general benchmarks once it has real data.
